# Simulation results

Raw output of the experiment sweeps reported in the paper (generated by
`../scripts/run_sweep.py` and `../scripts/fig_data_sweep.py`). Reviewers can
inspect these directly or regenerate them from a stock ns-3.48 tree.

| File | Contents |
|---|---|
| `results.csv` | main 3-scheduler sweep: regime, scheduler, seed, AoI/peak/duty (10 seeds) |
| `mixedmcs.csv` | mixed-modulation scaling (N=20,50; 3 schedulers) |
| `scaling.csv` | scaling N=10..50 (het. budget) |
| `valgrid.csv` | single-station model validation across per-SP loss |
| `validation.csv` | validation seed sweep (prediction error per seed) |
| `results_macros.tex`, `lb_macros.tex` | LaTeX macros (means + 95% CI, lower bounds) consumed by the paper |
| `figures/` | the generated plots (Figs. 3–7) |

Columns of `results.csv` / `mixedmcs.csv` / `scaling.csv`:
`aoi_ms` = weighted mean AoI, `peak_ms` = weighted mean peak AoI,
`duty` = mean station duty cycle. Regenerate the macros/figures with:

```bash
python3 ../scripts/make_macros.py results.csv results_macros.tex
python3 ../scripts/lower_bound.py 10 lb_macros.tex
python3 ../scripts/make_figs.py . figures
```
